🛠️ 消息推送凭证（Token/Webhook）获取指南

📬 1. 如何获取 Server酱 接收密钥？（微信接收） ▼

① 访问 Server酱 官网 并登录您的账号。

② 进入 「信道配置」，选择您期望的接收方式（如微信服务号、企业微信等）并完成关注绑定。

③ 进入 「SendKey」 页面，复制您的专属密钥（通常为 SCT 或 SCU 开头的英文字母与数字组合短串）。

④ 填写规范：请直接复制该密钥短串填入上方凭证输入框中，切勿携带任何 http:// 或 https:// 前缀。

🤖 2. 如何获取 钉钉群机器人 Webhook 地址？ ▼

① 打开钉钉电脑端，进入您需要接收通知的钉钉群聊。

② 点击右上角 「群设置」 -> 「智能群助手」 -> 「添加机器人」。

③ 选择 「自定义 (Custom)」 机器人，点击添加。

④ 安全设置（必填）：必须勾选 「自定义关键词」，并添加关键词：金价（否则机器人将拒绝投递消息）。

⑤ 完成后，复制系统生成的完整 Webhook 地址（带有 oapi.dingtalk.com 域名）。

⑥ 填写规范：可直接将整个完整 URL 链接粘贴到上方，系统后端会自动智能识别清洗出 Token 部分。

🕊️ 3. 如何获取 飞书群机器人 Webhook 地址？ ▼

① 打开飞书客户端，进入需要接收通知的飞书群聊。

② 点击右上角 「设置」 -> 「群机器人」 -> 「添加机器人」。

③ 选择 「自定义机器人」，点击添加并为其设置一个您喜欢的名称。

④ 安全设置（可选）：如需添加安全校验，建议使用包含关键词校验。

⑤ 完成后，复制系统生成的完整 Webhook 地址（带有 open.feishu.cn 域名）。

⑥ 填写规范：可直接将整个完整 URL 链接粘贴到上方，系统内置的清洗器会自动精准完成回显提取。
